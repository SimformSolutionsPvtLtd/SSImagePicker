# Way to contribute

1.  Fork the repo and create your branch from `master`.
2.  Clone the project to your own machine. (Please have a look at  [**Readme.md**](https://github.com/SimformSolutionsPvtLtd/SSImagePicker/blob/master/README.md) to understand how to run this project on your machine)
3.  Commit changes to your own branch
4.  Make sure your code lints.
5.  Push your work back up to your fork.
6.  Issue that pull request!
